# Tata-Sierra-Performance-Matrix-
This dataset simulates the end-to-end commercial lifecycle of the relaunched Tata Sierra (covering both its futuristic EV lineup and legacy-inspired ICE Petrol/Diesel variants) across a 15-month timeline (01-03-2025 to 24-05-2026). It utilizes a professional Star Schema database architecture designed to optimize Power BI's data modeling engine.
